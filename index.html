Infectious Disease Forecasting

A local infectious disease forecasting toolkit built with Python, Flask, SARIMA, Fourier seasonal modeling, neural-network forecasting, and ensemble prediction.

Important Deployment Note

This project is a Python/Flask application and is designed to run locally. GitHub Pages only supports static websites, so the full forecasting app cannot run directly on GitHub Pages.

Run Locally

git clone https://github.com/kmkhoury-15/infectious-disease-forecasting.git
cd infectious-disease-forecasting
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app/app.py

Then open:

http://127.0.0.1:5000
